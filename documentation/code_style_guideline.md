# Code Style Guideline

## Comments

- Complete Java/AppleDoc Comment:
  
  - Classes
  
  - Methods
  
  - Attributes
  
  - Functions

- Simple Comments for:
  
  - Loops
  
  - (Important Variables)

## Code Style

- Methods, Functions: camelCase

- Variables, Parameters, Attributes: snake_case

- Classes: PascalCase

## Other

- Take warnings serious and remove them

- Check parameters
  
  - if illegal, throw exception
